# Mesto — Проектная работа

Интерактивный сервис для обмена фотографиями мест. Пользователи могут добавлять фотографии, ставить лайки, удалять карточки и редактировать свой профиль.
Разработан вариант 2.

## Технологии

- HTML5, CSS3 (BEM)
- JavaScript (ES6+)
- Vite (сборщик)
- REST API

## Команды

```bash
# Установка зависимостей
npm install

# Запуск локального сервера разработки
npm run dev

# Сборка проекта
npm run build

# Публикация на GitHub Pages
npm run deploy
```
